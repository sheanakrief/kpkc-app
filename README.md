# kpkc-app

**KPKC.APP** — Hub Passeport Administratif Universel.

Page d'accueil de la plateforme avec accès aux 4 espaces :
- **PERSO** (KPARKPERSO) — Vie personnelle & administrative
- **IMMO** (KPARKIMMO) — Patrimoine immobilier → parkimmo.app
- **BUSINESS** (KPARKBUSINESS) — Gestion d'entreprise
- **PROTECT** (PARKPROTECT) — Protection, assurances, litiges, conformité

## Dev local

```bash
npm install
npm run dev   # → http://localhost:3004
```

Port 3004 pour éviter conflit avec les autres projets.

## Déploiement

Vercel → import `sheanakrief/kpkc-app` → domaine `kpkc.app`
